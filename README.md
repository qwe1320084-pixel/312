# Random Password Generator

Автор: Шаренко Богдан

## Описание

Простое графическое приложение для генерации случайных паролей с возможностью настройки:

- Длина пароля (от 4 до 32 символов)
- Использование цифр, букв, спецсимволов
- Копирование пароля в буфер обмена
- Сохранение истории паролей в JSON
- Просмотр истории в таблице

## Установка и запуск

1. Убедитесь, что установлен Python 3.6+
2. Запустите приложение:

```bash
python main.py
