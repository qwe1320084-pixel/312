Простое графическое приложение, предназначенное для конвертации российских рублей (RUB) в доллары США (USD), евро (EUR) и фунты стерлингов (GBP).

Курс обмена валют зафиксирован в коде программы и не обновляется в реальном времени.

Основные возможности
Удобный графический интерфейс (GUI).
Выпадающий список для выбора целевой валюты.
Встроенная проверка на корректность введённых данных.
Мгновенное отображение результата конвертации.
Технологии
Язык: Python 3.x.
Библиотека: tkinter (стандартная библиотека для создания GUI).
Как запустить
Убедитесь, что на вашем компьютере установлен Python 3.
Скопируйте файл converter.py (или название вашего файла) в папку проекта.
Откройте терминал или командную строку в этой папке.
Выполните команду:
bash
Копировать
python converter.py
В появившемся окне введите сумму и выберите валюту.
Примеры использования
Введите сумму в рублях и выберите валюту из списка. Нажмите кнопку «Конвертировать».

Ввод (RUB)	Валюта	Ожидаемый результат
1000	USD	10.80 USD
500	EUR	5.00 EUR
2500	GBP	23.00 GBP
